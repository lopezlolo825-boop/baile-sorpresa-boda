WEB BAILE SORPRESA - V0

Cómo conseguir el link público:
1. Entra en Netlify Drop.
2. Arrastra la carpeta "baile-sorpresa-web-v0" completa, no un archivo suelto.
3. Netlify dará un enlace terminado en .netlify.app.
4. Ese enlace ya se puede mandar por WhatsApp.

Para actualizar:
- Pide cambios en ChatGPT.
- Se genera una nueva versión de esta misma carpeta.
- En Netlify, entra en tu proyecto y arrastra la carpeta actualizada en Production deploys.
- El enlace puede seguir siendo el mismo si actualizas el mismo proyecto.

Archivos importantes:
- data.js: canciones, instrucciones y perfiles.
- assets/: imágenes de posiciones.
- index.html, styles.css, app.js: estructura de la web.
